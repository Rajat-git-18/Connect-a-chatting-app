# Coding Standards

## General

- TypeScript only
- Functional Components
- Hooks only
- No Class Components

## Naming

Components

PascalCase

Example

LoginScreen.tsx

Hooks

camelCase

Example

useAuth.ts

Folders

lowercase

Constants

UPPER_CASE

Interfaces

Prefix with I

Example

IUser

Enums

PascalCase

Functions

camelCase

Variables

camelCase

Reusable components
